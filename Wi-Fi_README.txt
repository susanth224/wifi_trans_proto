Wi-FileTransproto is a Android Mobile Application to share the files to the favourite connected devices.
Software Requirements used to develop this Application.
1.Android SDK it requires JDK 5 or Higher we use JDK7.
2.Eclipse 3.5
2.ADT(Android Development Tools)
3.Android OS
4.SQLite(Structured Query Language with Lightweight database access and control )
5.Java
6.Java eclipse
7.J2ME(Java 2 Mobile Enterprise)
8.XML(Extensive Markup Language)
open the .apk created in ADT with my project name is copied to the Android mobile.
Now open  that file it will scan the Wi-Fi connected devices then we will share the file to the favourite connected devices.
